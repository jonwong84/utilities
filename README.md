# utilities

A collection of utilities for my projects.

## Utilities.Configuration

A utility for managing configuration settings in my projects. It provides a simple interface for loading, saving, and accessing configuration data, making it easier to manage application settings across different environments. The utility supports various formats such as JSON, YAML, and INI, allowing for flexibility in how configuration data is stored and accessed. It also includes features for validating configuration data and handling default values, ensuring that applications can run smoothly even when certain settings are missing or incorrect.
